# Usage 

import {
  Input,
  Label,
  Select,
  Checkbox
} from "@/components/ui/form"
export function SignupForm() {
  return (
    <form>
      <div class="grid gap-2">
        <Label htmlFor="email">Email</Label>
        <Input id="email" type="email" placeholder="m@example.com" />
      </div>
      <Button type="submit">Sign Up</Button>
    </form>
  )
}

API Reference

Standard HTML attributes are supported for all form elements.

Prop Type Default Description
variant "default" | "error" "default" Controls the border color and focus states.
size "sm" | "md" | "lg" "md" The height and padding of the input field.
disabled boolean false Disables interaction and dims opacity.
