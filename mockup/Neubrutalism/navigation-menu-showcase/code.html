Navigation Menu

A collection of links for navigating websites. Flexible, responsive, and styled with zero compromises. Supports nested lists and complex content.

# Installation

terminal
npm install @neubrutal/navigation-menu

# Usage 

import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuList,
  NavigationMenuTrigger,
} from "@/components/ui/navigation-menu"
export function NavigationMenuDemo() {
  return (
    <NavigationMenu>
      <NavigationMenuList>
        <NavigationMenuItem>
          <NavigationMenuTrigger>Item One</NavigationMenuTrigger>
          <NavigationMenuContent>
            <NavigationMenuLink>Link</NavigationMenuLink>
          </NavigationMenuContent>
        </NavigationMenuItem>
      </NavigationMenuList>
    </NavigationMenu>
  )
}

API Reference

The navigation menu is built on top of Radix UI primitives, ensuring full keyboard navigation and screen reader support.

Prop Type Default Description
defaultValue string - The value of the menu item that should be active by default.
delayDuration number 200 The duration from when the mouse enters a trigger until the content opens.
orientation "horizontal" | "vertical" "horizontal" The orientation of the menu.
dir "ltr" | "rtl" "ltr" The reading direction of the menu.
