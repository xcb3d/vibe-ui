Hover Card

A sighted user preview that reveals related content when hovering over a link or element, enhancing context without leaving the page.

# Installation

terminal
npm install @neubrutal/hover-card

# Usage 

import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "@/components/ui/hover-card"
export function HoverCardDemo() {
  return (
    <HoverCard>
      <HoverCardTrigger>Hover me</HoverCardTrigger>
      <HoverCardContent>
        The React Framework – created and maintained by @vercel.
      </HoverCardContent>
    </HoverCard>
  )
}

API Reference

Customize the behavior and appearance of the hover card.

Prop Type Default Description
openDelay number 700 The duration in ms to wait before opening.
closeDelay number 300 The duration in ms to wait before closing.
side "top" | "right" | "bottom" | "left" "bottom" The preferred side of the trigger to render against.
align "start" | "center" | "end" "center" The preferred alignment against the trigger.
