Avatar

An image element with a fallback for representing the user. Bold, bordered, and built to stand out.

API Reference Source Code
JD
person

# Installation

terminal
npm install @neubrutal/avatar

# Usage 

import {
  Avatar,
  AvatarFallback,
  AvatarImage,
} from "@/components/ui/avatar"
export function AvatarDemo() {
  return (
    <Avatar>
      <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
      <AvatarFallback>CN</AvatarFallback>
    </Avatar>
  )
}

API Reference

The avatar component is designed to be simple yet visually striking, supporting various fallback strategies.

Prop Type Default Description
src string - The source URL of the avatar image.
alt string - Alt text for the avatar image.
asChild boolean false Change the default rendered element for the one passed as a child.
