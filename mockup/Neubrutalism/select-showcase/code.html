Select

Displays a list of options for the user to pick from—triggered by a button. Built bold for clear choices.

# Installation

terminal
npm install @neubrutal/select

# Usage 

import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@/components/ui/select"
export function SelectDemo() {
  return (
    <Select>
      <SelectTrigger className="w-[180px]">
        <SelectValue placeholder="Theme" />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="light">Light</SelectItem>
        <SelectItem value="dark">Dark</SelectItem>
        <SelectItem value="system">System</SelectItem>
      </SelectContent>
    </Select>
  )
}

API Reference

The select component is built for tough interfaces, mimicking native behavior but with custom styling capabilities.

Prop Type Default Description
value string - The controlled value of the select item.
onValueChange function - Event handler called when the value changes.
defaultValue string - The default value when initially rendered.
disabled boolean false Prevents interaction with the select component.
