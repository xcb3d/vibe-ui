Pagination

Navigation for breaking up large datasets or content into manageable pages. Bold, clear, and unmissable.

API Reference Source Code
Prev 1 2 3 ... 12 Next

# Installation

terminal
npm install @neubrutal/pagination

# Usage 

import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationNext,
  PaginationPrevious,
} from "@/components/ui/pagination"
export function PaginationDemo() {
  return (
    <Pagination>
      <PaginationContent>
        <PaginationItem>
          <PaginationPrevious href="#" />
        </PaginationItem>
        <PaginationItem>
          <PaginationLink href="#">1</PaginationLink>
        </PaginationItem>
        <PaginationItem>
          <PaginationLink href="#" isActive>2</PaginationLink>
        </PaginationItem>
        <PaginationItem>
          <PaginationNext href="#" />
        </PaginationItem>
      </PaginationContent>
    </Pagination>
  )
}

API Reference

The pagination component is built with composability in mind, allowing you to construct complex paging layouts.

Prop Type Default Description
count number 1 The total number of pages.
page number 1 The current active page.
siblingCount number 1 Number of always visible pages before and after the current page.
onPageChange function - Callback fired when the page changes.
