Table

A powerful, responsive data table component for displaying rows of data efficiently. Built with bold borders for high contrast and readability.

# Installation

terminal
npm install @neubrutal/table

# Usage 

import {
  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableHead,
  TableHeader,
  TableRow,
} from "@/components/ui/table"
export function TableDemo() {
  return (
    <Table>
      <TableCaption>A list of your recent invoices.</TableCaption>
      <TableHeader>
        <TableRow>
          <TableHead class="w-[100px]">Invoice</TableHead>
          <TableHead>Status</TableHead>
          <TableHead>Method</TableHead>
          <TableHead class="text-right">Amount</TableHead>
        </TableRow>
      </TableHeader>
      <TableBody>
        <TableRow>
          <TableCell class="font-medium">INV001</TableCell>
          <TableCell>Paid</TableCell>
          <TableCell>Credit Card</TableCell>
          <TableCell class="text-right">$250.00</TableCell>
        </TableRow>
      </TableBody>
    </Table>
  )
}

API Reference

The table component is composed of several sub-components to handle layout and structure.

Component Prop Type Description
Table - - The main wrapper for the table.
TableHead align "left" | "center" | "right" Aligns the header content.
TableRow onClick function Event handler for row clicks (useful for selection).
