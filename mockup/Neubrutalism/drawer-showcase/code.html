Drawer

A panel that slides out from the edge of the screen. Can be used for navigation, details, forms, or any side content that needs attention.

API Reference Source Code

Interact with the component

# Installation

terminal
npm install @neubrutal/drawer

# Usage 

import {
  Drawer,
  DrawerContent,
  DrawerTrigger,
  DrawerClose,
} from "@/components/ui/drawer"
export function DrawerDemo() {
  return (
    <Drawer direction="right">
      <DrawerTrigger>Open</DrawerTrigger>
      <DrawerContent>
        <DrawerHeader>
          <DrawerTitle>Are you sure?</DrawerTitle>
        </DrawerHeader>
        <DrawerFooter>
          <DrawerClose>Cancel</DrawerClose>
        </DrawerFooter>
      </DrawerContent>
    </Drawer>
  )
}

API Reference

The drawer uses the dialog primitive and supports full keyboard navigation and focus management.

Prop Type Default Description
direction "left" | "right" | "top" | "bottom" "right" The direction from which the drawer slides in.
open boolean false The controlled open state of the drawer.
dismissible boolean true Whether clicking outside or pressing Escape closes the drawer.
