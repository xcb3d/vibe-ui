Label

Renders an accessible label associated with controls. Essential for forms, indicators, and providing context with style.

API Reference Source Code

# Installation

terminal
npm install @neubrutal/label

# Usage 

import { Label } from "@/components/ui/label"
export function LabelDemo() {
  return (
    <div>
      <div class="flex items-center space-x-2">
        <Checkbox id="terms" />
        <Label htmlFor="terms">Accept terms and conditions</Label>
      </div>
    </div>
  )
}

API Reference

The Label primitive is based on standard HTML label element but styled for consistency.

Prop Type Default Description
htmlFor string - The id of the element the label is associated with.
className string - Additional CSS classes for custom styling.
